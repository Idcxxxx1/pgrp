Privacy Policy

Last updated: July 14, 2025

Introduction
This Privacy Policy explains how PGRP Bot ("we", "our", or "the bot") collects, uses, stores, and protects information from users on Discord servers where the bot is invited and used.

1. Information We Collect

  • Discord User Data: User IDs, usernames, and interaction logs for command usage.
  • Vehicle Registration Data: Custom user-submitted vehicle information including license plates, make, model, year, and color.
  • Tickets & Support Logs: User-created tickets with message transcripts, claimed by information, and session logs.
  • Command Logs: Usage records for internal diagnostics (e.g., session starts, co-hosts, reinvites).
  • MySQL Data: All aforementioned data is stored securely in a MySQL database.

2. How We Use This Data

  • To provide bot functionalities (commands, automations, ticket systems).
  • To generate session logs and vehicle/ticket profiles.
  • To maintain security and moderation logs.

3. Data Storage & Security
All data is stored securely in a private MySQL database, protected by authentication and access restrictions. The bot is hosted on Cybrancee, which provides server-side security controls.

4. Data Sharing
We do not share user data with any third parties. All information is used solely for internal bot operations within the PGRP server.

5. Data Retention
Data such as vehicle registrations and ticket logs are retained until explicitly deleted by a moderator or command (e.g., `/unregister`, `/ticketremove`).

6. User Rights
Users can request deletion of their data by contacting the PGRP server moderators or submitting a ticket in the appropriate channel.

7. Changes to This Policy
We may update this Privacy Policy as the bot grows. Users will be notified of significant changes via Discord announcements.

Contact
For privacy concerns or questions, open a support ticket or contact @idcxxxx1 directly in the PGRP Discord server.
